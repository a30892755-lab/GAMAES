Flexbox: Layout Ka Zabardast Tareeqa

Flexbox ek ek-tarafah (one-dimensional) layout model hai, jiska maqsad hai ki aap apne web page ke components ko aasani se arrange, space, aur align kar saken. Is interactive tool (Sandpit) mein, aap in zaroori properties ko badal kar unke asar ko foran (instantly) dekh sakte hain.

Flex Container (Baap Element) - `display: flex;`

Yeh woh bada dabba hai jis par hum `display: flex;` lagate hain. Yeh apne andar ke saare bachchon (items) ka layout control karta hai.

Flex Items (Bachche Elements)

Yeh woh chhote dabba hain jo container ke andar hote hain. In par `flex-grow` aur `order` jaisi properties lagti hain.

3. Sizing Control (`flex-grow`)

`flex-grow` batata hai ki container mein bachi hui extra jagah ko items aapas mein kaise baantenge. Neeche values daalen aur dekhein ki Items kaise stretch hote hain. Item ki size uske `flex-grow` value ke mutabiq badhegi.

4. Zaroori Properties (Quick Reference)

Yahan Flexbox ki do main kirdaron (Container aur Items) par lagne wali zaroori properties ki ek mukammal list hai. Click karke detail dekhiye.

Property Matlab (Meaning)
display: flex; Container ko Flexbox bana do. Sabse pehla step.
flex-direction Items kis disha (direction) mein jayenge: `row` ya `column`.
justify-content Main Axis par items ko kaise baantna hai (Horizontal spacing).
align-items Cross Axis par items ko kaise align karna hai (Vertical alignment).
Property Matlab (Meaning)
flex-grow Agar container mein extra jagah hai, toh yeh item kitni jagah 'khaayega'.
flex-shrink Agar jagah kam hai, toh yeh item kitna chhota hoga.
flex-basis Item ki shuruwaati (default) size kya hogi.
order Items ko kis tarteeb (sequence) mein dikhana hai.
